HASHIRA AI
Hashira AI es un asistente virtual inteligente diseñado para entornos educativos, enfocado en el bienestar estudiantil y la detección de emociones, desarrollado con tecnología de punta para brindar soporte personalizado.

Tecnologías utilizadas
Frontend: Flutter/Dart (Desarrollo móvil multiplataforma).

Backend: Firebase (Gestión de base de datos y autenticación en tiempo real).

Inteligencia Artificial: Integración con la API de Gemini para procesamiento de lenguaje natural y soporte al estudiante.

Configuración del proyecto
Para ejecutar este proyecto localmente, asegúrate de realizar los siguientes pasos:

Clonar el repositorio:

Bash
git clone https://github.com/ALHEX-222/HASHIRA.git
Configurar variables de entorno:
Crea un archivo .env en la raíz del proyecto y añade tu clave de API:

Plaintext
API_KEY=tu_api_key_aqui
Firebase:

Asegúrate de colocar tus archivos google-services.json (Android) y GoogleService-Info.plist (iOS) en sus respectivas carpetas dentro de android/app/ y ios/Runner/.

Estado del proyecto
